LECTURE RAPIDE

- Azure Blob Storage centralise les logs et les resultats.
- Azure Function Timer lance automatiquement le pipeline.
- Le pipeline Python prepare les donnees et detecte les anomalies.
- Le dashboard Streamlit lit les resultats depuis Azure et les affiche.

MESSAGE DU PROJET

Cette architecture represente la version finale cloud du projet:
- donnees stockees dans Azure
- traitement automatise dans Azure
- visualisation en ligne avec Streamlit
